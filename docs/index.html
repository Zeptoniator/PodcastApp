Politique de confidentialité

PodcastApp

Dernière mise à jour : 04/03/2026

Cette politique décrit la manière dont PodcastApp traite vos données lors de l'utilisation de l'application.

Données collectées

Selon les fonctionnalités utilisées, l'application peut traiter :

  • des données de compte (ex. e-mail) pour l'authentification ;
  • des données de profil et préférences (abonnements, paramètres) ;
  • des contenus synchronisés que vous choisissez d'enregistrer.

Utilisation des données

Les données sont utilisées pour fournir les fonctionnalités de PodcastApp, maintenir votre session, synchroniser vos contenus et améliorer la fiabilité du service.

Stockage et sécurité

Les données peuvent être stockées localement sur l'appareil et/ou dans des services cloud associés à l'application (ex. Firebase) afin d'assurer la continuité d'utilisation.

Des mesures techniques raisonnables sont appliquées pour protéger les données contre l'accès non autorisé.

Partage des données

Aucune vente de données personnelles n'est effectuée. Les données ne sont partagées qu'avec les sous-traitants techniques nécessaires au fonctionnement du service.

Vos droits

Vous pouvez demander l'accès, la correction ou la suppression de vos données, selon les obligations légales applicables.

Contact

Pour toute question relative à la confidentialité : support@podcastapp.com

Modifications

Cette politique peut évoluer. Toute mise à jour sera reflétée par la date indiquée en haut de la page.

Vérifiez ce document régulièrement pour rester informé des changements.
